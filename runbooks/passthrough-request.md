# Passthrough Request (Cross-Module / Cross-Team)

## When to use
Use when a module change is required outside the requesting PM’s ownership.

## Required artifacts
- Problem Statement (PM-owned)
- Acceptance Criteria (PM-owned)
- Risk Assessment (initial)
