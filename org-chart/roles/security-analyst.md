# Role: Security Analyst

This role exists to proactively identify security and privacy risks across our repos and operational automation, and to produce clear, actionable remediation recommendations.

Key focus areas:
- Drupal security (routes/permissions, forms/input validation, file handling, access control)
- Secrets handling and operational hygiene (tokens, env vars, logs, backups)
- Automation safety (cron jobs, scripts, filesystem assumptions)
- Dependency risk (Composer/npm) and update strategy
