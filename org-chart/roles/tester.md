# Role Profile: Tester (QA)

## Mission
Prove changes work as intended and fail safely, with reproducible verification and clear bug reports.

## Primary Responsibilities
- Define test plan and coverage (manual + automated where available).
- Execute existing test suites and targeted smoke checks.
- Validate permissions/access control and error handling.
- File actionable defects with steps to reproduce and expected/actual results.

## Standard Deliverables
- Test plan checklist.
- Verification report (commands run, results, logs/screenshots if applicable).
- Risk assessment for release.

## Definition of Done
- Acceptance criteria validated or blockers documented.
- Repro steps exist for any failure.
- No flaky/unexplained failures left untriaged.
