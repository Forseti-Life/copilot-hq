# Features / Work Items

Each folder under `features/` is a trackable work item.

## Rules
- Keep one canonical `feature.md` per work item.
- Updates go in the "Latest updates" section and/or linked artifacts under `sessions/<agent>/artifacts/`.
- Active features should include `- Project: PROJ-*` so backlog work is explicitly linked to the CEO portfolio registry in `dashboards/PROJECTS.md`.
